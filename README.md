##Files for my QuickSort test

- Sources
- Shell script for timing
- Checking program
- CSV writing script

####Instructions for Running the Test

0. Download the repo
0. Make Sure you have the dependencies:
    - `make`
    - `gcc` & `g++`
    - `jdk`
    - `php5-cli`
    - `python3`
    - `pypy`
    - `jython`
0. Run the test
    - `cd cmplang`
    - `sh setup.sh`
    - `./TestTime.sh`
0. The results can be found at `cmplang/results/RunTimesTable.csv`
